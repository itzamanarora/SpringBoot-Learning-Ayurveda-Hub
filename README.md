## Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/itzamanarora/SpringBoot-Learning-Ayurveda-Hub.git
   ```
2. **Navigate to the project directory:**
   ```bash
   cd SpringBoot-Learning-Ayurveda-Hub
   ```
3. **Install dependencies & build:**
   ```bash
   mvn install
   # or
   gradle build
   ```
4. **Run the application:**
   ```bash
   mvn spring-boot:run
   # or
   gradle bootRun
   ```

## Configuration

- Modify `application.properties` or `application.yml` in `src/main/resources` for local setup.
- Set your database, server port, and other properties as needed.
